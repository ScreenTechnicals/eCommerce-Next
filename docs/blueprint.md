# **App Name**: eCommerce Next

## Core Features:

- Home Page: Display a landing page with a hero section, category highlights, new arrivals and a simple newsletter sign-up form.
- Product Listings: Catalog page with product listings including filtering, sorting, and pagination to improve browsing. All displayed information (product name, pricing) is rendered from dummy data.
- Product Detail Page: Detailed product pages, populated from the product id found in the route, showcasing product details such as high-resolution images, description, and price, with 'Add to Cart' functionality.
- Shopping Cart: A cart page that dynamically renders added items. Users are able to adjust quantities and remove items before navigating to a basic checkout process.
- Checkout Page: Streamlined checkout process that confirms order details and simulates submission, enhancing the shopping experience.
- User Authentication: Simple login and registration components, without database connectivity, for basic user authentication simulation using local storage to persist user sessions.
- AI-Powered Product Search: A search bar, using a tool that considers relevancy of matches, allowing for quick filtering of products based on user input.

## Style Guidelines:

- Primary color: Indigo (#667EEA) for a modern, trustworthy feel.
- Background color: Off-white (#F7FAFC), providing a clean, neutral backdrop.
- Accent color: Purple (#9F7AEA), for highlights and interactive elements, creating visual interest.
- Body font: 'Inter' sans-serif for a clean, modern reading experience. Headline font: 'Space Grotesk' for bold headlines.
- Lucide icons, providing a consistent, minimalist aesthetic.
- Grid-based layout with responsive breakpoints for optimal viewing across devices.
- Subtle transitions and hover effects to enhance interactivity and user experience.
